# Steno Only

For use with [Plover](https://github.com/openstenoproject/plover)

Compile with `make preonic/rev3:steno`

### Steno

```
 * ,-----------------------------------------------------------------------------------.
 * |      |      |      |      |      |      |      |      |      |      |      |      |
 * |------+------+------+------+------+------+------+------+------+------+------+------|
 * |   S  |   T  |   P  |   H  |   *  |      |   *  |   F  |   P  |   L  |   T  |   D  |
 * |------+------+------+------+------+-------------+------+------+------+------+------|
 * |   S  |   K  |   W  |   R  |   *  |      |   *  |   R  |   B  |   G  |   S  |   Z  |
 * |------+------+------+------+------+------|------+------+------+------+------+------|
 * |      |      |      |      |      |      |      |      |      |      |      |      |
 * |------+------+------+------+------+------+------+------+------+------+------+------|
 * |      |      |   #  |   A  |   O  |      |   E  |   U  |   #  |      |      |      |
 * `-----------------------------------------------------------------------------------'
```
